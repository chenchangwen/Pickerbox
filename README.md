关于pickerbox

pickerbox简称采摘盒子,基于Jquery的插件,引用了JQueryzTree-v3.5.12,easyui1.3.2,及ColorBox v1.3.20.1.只为更方便的取得简单的或多条件的数据.

-支持树,表格呈现数据.

-支持跨iframe获取数据.

-灵活支持记录数据.

-支持重要事件回调.


支持浏览器

成功测试在以下浏览器

Chrome

Firefox 3.5+

Safari 3.2.2+

Opera 10.6+

IE6,7,8
